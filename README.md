### Project Overview

AI Career Assistant is a Streamlit-based application designed to help users with career preparation. The application generates resume summaries, cover letters, interview questions, and skill recommendations based on user inputs.

This project demonstrates the deployment of a GenAI-style application on Streamlit Cloud and Hugging Face Spaces.


## Features

### 1. Resume Summary Generator

Generates a professional resume summary based on:

* Name
* Job Role
* Experience
* Skills

### 2. Cover Letter Generator

Creates a professional cover letter for job applications.

### 3. Interview Questions Generator

Provides commonly asked interview questions based on the selected job role.

### 4. Skill Suggestions

Recommends additional skills that can improve career opportunities.
